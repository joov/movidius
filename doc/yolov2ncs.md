# Setup YoloV2NCS

## Requires

- [Intel Movidius NCS USB stick]
- The SDK only supports 

## Steps

1. Install [NCS SDK](https://github.com/movidius/ncsdk)

> Remember to use `-b ncsdk2` in `git clone` so you won't get old v1 SDK.

2. Build YoloV2NCS

```console
$ cd yolo3-camera/vendor/YoloV2NCS
$ make -j4
```
